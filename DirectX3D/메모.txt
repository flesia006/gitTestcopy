2. 레스터라이저 스테이트
	- 정점화된 텍스쳐(2D 이미지)를 픽셀로 변환해서 출력하는 절차
	- 텍스처 표현 + 3D 모델을 모니터에서 보기 위한 그림으로 변환한다.
	- 대표적인 기능 : 
		CullMode 선택 -> D3D11_CULL_BACK : 출력 선별 기준, 기본값 _ 뒷면은 버린다
		FILLMODE 선택 -> D3D11_FILL_SOLID : 텍스쳐 출력 기준, 기본값 _ 다 그린다

	- 추가로 FrontCounterClockWise :  전면 텍스쳐가 시계인지 반시계인지 선택.

3. 블랜드 스테이트
	- 의미 1 : 이미지에 등록되었거나, 외부에서 직접 설정한 화소를 탈락시키거나(배경색 날리기)
	- 의미 2 : 같은 위치에 있는 이미지와 화소를 합치거나 (반투명 처리)
	- 의미 3 : 이미지를 다른 곳으로 옮겨서 투사하는 절차 담당

	대표적인 기능 : RenderTargetWriteMask :  불투명 이미지가 겹칠 때 투과 여부를 결정!!!
					기본값 :  D3D11_COLOR_WRITE_ENABLE_ALL "얘 보다 앞에 있는 것은 다 마스크다! 앞에 있는 물체가 뒤 물체를 가린다"
							D3D11_COLOR_WRITE_ENABLE_NONE : RenderTargetWriteMask 옵션 설정에서 별도 접근권이 필요없이
															호출가능한 두 옵션.
															뜻은 어떤 것도 이미지를 가리지 않음 (뒤에 있는 그림도 다 보임!)

4. 뎁스 스텐실 스테이트
	- 이미지의 우선순위 (z 축 좌표 혹은 강제출력순위) 설정 및 (뎁스)
	- 이미지를 가려서 못 보이게 하는 조건 여부와 세부 설정 (스텐실)
	- 위의 둘을 합쳐 이미지의 선별적 출력을 담당하는 스테이트

	대표적인 기능 :  깊이 사용할거? 깊이 판단 시 우선순위, 깊이 판단 기준(기본은 카메라와의 거리).
